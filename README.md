# Begriff

> Begriffsschrift suele traducirse al español como Conceptografía (basado en un libro sobre lógica escrito por Gottlob Frege); el título completo del libro lo identifica como «un lenguaje formal, modelado según el de la aritmética, para el pensamiento puro»

Esta wiki/common-place refleja intereses e investigaciones idiosincrásicas.
